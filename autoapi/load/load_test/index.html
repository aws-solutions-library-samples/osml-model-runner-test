load.load_test.start_workflow() → None

The workflow to build an image request for a specific SM endpoint and then place it on the corresponding SQS queue every defined second for ModelRunner to pick up and process. Once the image has been completed, return the associated image_id and image_request object for analysis. Then it will display the results for all the images have been sent for ModelRunner and calculate how many have succeeded or failed.

Returns:

None
